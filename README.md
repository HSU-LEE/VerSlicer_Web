# VerSlicer Web

Official website for [VerSlicer](https://github.com/HSU-LEE/VerSlicer) — an AI-integrated macOS slicer built on OrcaSlicer.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy (GitHub Pages)

Push to `main` triggers [.github/workflows/deploy.yml](.github/workflows/deploy.yml).

Site URL: `https://hsu-lee.github.io/VerSlicer_Web/`

In the [VerSlicer](https://github.com/HSU-LEE/VerSlicer) repository, set **About → Website** to the URL above so users can return from GitHub.
